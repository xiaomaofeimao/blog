# 第三章: 小试牛刀

这一章，我们将尝试完成下图的页面：卡片列表页。完整的 demo 代码请访问 https://github.com/ant-design/react-tutorial 。

在我们的技术栈中 React 负责的是将数据渲染成页面，但是并不负责管理数据本身。对于一个单页应用而言，数据的管理会是一个关键点，页面 + 数据才构成一个完整的前端应用。本章我们会一步一步讲述如何在页面中管理数据。
